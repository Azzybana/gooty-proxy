Expand description

Fetch proxies from multiple sources concurrently.

This function scrapes proxies from all provided sources in parallel, applying rate limiting and error handling.

§Arguments

  • sources - Slice of Source objects to fetch proxies from
  • requestor - The Requestor instance to use for HTTP requests
  • concurrency - Maximum number of concurrent fetch operations

§Returns

A vector of unique proxies fetched from all sources.

§Errors

Returns an error if there’s a critical failure in the fetch process. Individual source failures are logged but don’t cause the entire operation to fail.
