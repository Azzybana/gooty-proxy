Expand description

Main Sleuth struct for performing IP lookups

The Sleuth service provides comprehensive IP intelligence by querying various data sources to gather information about IP addresses. It can retrieve hostname, network, location, and ownership information.

§Examples

use gooty_proxy::inspection::Sleuth;
use std::net::{IpAddr, Ipv4Addr};

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    // Create a new Sleuth instance
    let sleuth = Sleuth::new();

    // Look up information for Google's DNS server
    let ip = IpAddr::V4(Ipv4Addr::new(8, 8, 8, 8));
    let hostname = sleuth.lookup_hostname(&ip).await?;

    println!("Hostname: {:?}", hostname);

    Ok(())
}
